c. Project Description: A private repository containing the source file of our OOP group's game project, Sentinel, which a strategy-based warfare simulation that can run in cmd.

d. System Features:
- Pick 1 of 50 countries, manage MIL / ECO / POP / STABILITY
- Launch attacks, send aid, or request support from 2 random allies
- Random global news events shake up every session (crashes, outbreaks, buildups)
- AI countries war with each other in the background independently
- Beat your objective before the 24-hour in-game clock runs out, or lose

e. OOP Concepts Used:
Encapsulation
inheritance
polymorphism
